# 🛠️ DafianTools — Web Downloader TikTok & Instagram

![DafianTools](https://img.shields.io/badge/Version-1.0-blue)
![License](https://img.shields.io/badge/License-MIT-green)
![Deploy](https://img.shields.io/badge/Deploy-Vercel-black)

**DafianTools** adalah web tools gratis buat download video TikTok tanpa watermark & download foto/video/reels/story Instagram HD. Dibangun dengan vanilla HTML/CSS/JS, ringan, dan siap deploy di Vercel.

---

## ✨ Fitur

### 🎵 TikTok Tools
- Download Video HD (No Watermark)
- Download Slide Foto
- Download MP3 Audio
- Download Story
- Download Foto Profil HD
- Download Thumbnail

### 📸 Instagram Tools
- Download Foto HD
- Download Galeri (Multi Foto)
- Download Story IG
- Download Reels Video
- Download Video IG

### 🎨 UI/UX
- Animasi smooth (fade, slide, float)
- Popup sambutan (pagi/siang/sore/malam/subuh)
- Jam real-time
- Mobile responsive
- Loading animations
- Toast notifications
- Dark theme glassmorphism

---

## 🚀 Deploy ke Vercel

### Cara 1: GitHub Import
1. Fork / Clone repo ini
2. Buka [vercel.com/new](https://vercel.com/new)
3. Import repo `dafian01/WebDos`
4. Klik Deploy!

### Cara 2: Vercel CLI
```bash
npm i -g vercel
git clone https://github.com/dafian01/WebDos.git
cd WebDos
vercel --prod
